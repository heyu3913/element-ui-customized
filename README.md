## 1、介绍
本项目是基于element-ui@2.15.14的基础上，二次修改源码，封装的ui组件，在此特别感谢[elemen-ui组件库](https://github.com/ElemeFE/element)

## 2、更新功能

![img.png](img.png)
#### 使用方式：
* 下载依赖：
```javascript
 cnpm i @heyu3913/element-ui-customized -S
```
* 可以在main.js中按需使用:
```javascript
import Vue from 'vue'
import App from './App.vue'
import router from './router'
import { Cascader } from '@heyu3913/element-ui-customized';
import '@heyu3913/element-ui-customized/theme-chalk/index.css'; //很重要，一定要引入

Vue.use(Cascader)
Vue.config.productionTip = false

new Vue({
  render: h => h(App),
  router,

}).$mount('#app')
```

  配置如下：
```javascript
      props: { multiple: true,onlyLastMultiple:true },
```
